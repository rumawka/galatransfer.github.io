# Gala Transfer — Инструкция по установке

## Структура файлов
```
gala-transfer/
├── index.html          ← главная страница
├── css/
│   └── style.css       ← все стили
├── js/
│   └── main.js         ← весь JavaScript
└── api/
    └── send.php        ← бэкенд (токен Telegram скрыт здесь)
```

## Загрузка на хостинг (Beget / Timeweb / Hostinger / любой с PHP)

1. Подключитесь к хостингу через **FTP** или **файловый менеджер** в панели управления
2. Зайдите в папку `public_html` (или `www`, или `htdocs` — зависит от хостинга)
3. Загрузите все файлы, сохраняя структуру папок
4. Готово — сайт работает

## Подключение домена

1. Купите домен (reg.ru, nic.ru, namecheap.com и др.)
2. В настройках домена укажите **NS-серверы** вашего хостинга
   (хостинг пришлёт их в письме при регистрации)
3. Подождите 1-24 часа — домен заработает

## Важно про API-токен

Токен Telegram спрятан в файле `api/send.php`.
Браузер его не видит — он исполняется только на сервере.

Если хотите сменить токен или chat_id — откройте `api/send.php`
и замените строки:
```php
define('TG_TOKEN',   'ВАШ_ТОКЕН');
define('TG_CHAT_ID', 'ВАШ_CHAT_ID');
```

## Если хостинг без PHP (Netlify / Vercel / GitHub Pages)

На таких хостингах `api/send.php` не работает.
Для них нужна другая схема — напишите, помогу настроить.
